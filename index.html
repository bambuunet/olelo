What is Olelo

  1. Markdwon file is displayed as HTML on browser.
  2. Called by only 1 sentense in script.
  3. Generated nest structure.
  4. JavaScript file is only 14KB.

Markdown

#### Olelo is...
- olelo.js display markdown file as HTML on browser.

Getting started

Download

Download Github

Preparing markdown

Write in a markdown file.
Rule: [ HTML tag name ] [ 1 space ] [ free text ]

h4 This is a headline.
p This is a paragraph.

Setting in HTML

Call the markdown file like this.
1st argument: Markdown file path
2nd argument: Id name in HTML that you want to display

<div id="tag_id"></div>

<script>
  Olelo("markdown.md", "tag_id");
</script>

Generated HTML

<h4>This is id sample.</h4>
<p>This is a paragraph.</p>

Nest

Unify the space number of indents with a certain numerical multiple.

Markdown

table
  tr
    th ID
    th Name
    th Date
  tr
    td 123
    td Ringo Takemura
    td 2018-05-01

Generated HTML

<table>
  <tr>
    <th>ID</th>
    <th>Name</th>
    <th>Date</th>
  </tr>
  <tr>
    <td>123</td>
    <td>Ringo Takemura</td>
    <td>2018-05-01</td>
  </tr>
</table>

With attribution

Markdown

p#my_id This is id sample.

p.my_class This is class sample.

p.class1.class2.class3 This is multiple class sample.

a(href="#"target="_blank") This is other attribution sample.

p#my_id.my_class(style="color:#ff0000") This is combined attribution sample.

Generated HTML

<p id="my_id">This is id sample.</p>

<p class="my_class">This is class sample.</p>

<p class="class1 class2 class3">This is multiple class sample.</p>

<a href="#" target="_blank">This is other attribution sample.</a>

<p id="my_id" class="my_class" style="color:#ff0000">This is combined attribution sample.</p>

On browser

Tag in line

Markdown

p Write [u an underline].

p Push [input(type="button"value="the button")].

p Click [a#my_id.my_class(href="#")the link].

Generated HTML

<p>Write <u>an underline</u>.</p>

<p>Push <input type="button" value="the button">.</p>

<p>Click <a id="my_id" class="my_class" href="#">the link</a>.</p>

Div

<div> is generated if the first character is "#" or ".".

Markdown

.class div block

<div class="class">.class div block</div>

Comment

Markdown

//This is visible comment
  1. visible
  2. visible

p Next is visible comment //This is visible comment

//-This is invisible comment
  a. invisible
  b. invisible

p Next is invisible comment //-This is invisible comment

Generated HTML

<!--
  This is visible comment
  1. visible
  2. visible
-->

<p>Next is visible comment<!-- This is visible comment --></p>

<p>Next is invisible comment</p>
